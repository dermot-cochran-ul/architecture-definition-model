This page is normative and defines ADM semantics.

# Context

## Definition
External actors, users, and systems.

## Allowed Content
- External actors
- User roles
- External systems

## Forbidden Content
- Internal component decomposition
- Deployment structure
- Implementation technology

## Common Violations
- Modeling internal components as context actors
- Introducing runtime internals in Context
