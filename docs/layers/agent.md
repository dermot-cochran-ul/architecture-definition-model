This page is normative and defines ADM semantics.

# Agent

## Definition
Autonomous behaviour or control logic (not structural).

## Allowed Content
- Autonomous behavior definitions
- Control logic boundaries
- Behavioral references to ADM elements

## Forbidden Content
- Structural decomposition of architecture
- New structural layers
- Deployment topology definitions

## Common Violations
- Using Agent to define architecture structure
- Treating Agent as a vertical structural layer
