This page is normative and defines ADM semantics.

# Capability

## Definition
What must be possible (non-executable).

## Allowed Content
- Business or system capabilities
- Non-executable ability statements
- Capability boundaries

## Forbidden Content
- Running services
- Deployable units
- Technology bindings

## Common Violations
- Representing executable code as Capability
- Describing deployment behavior as Capability
