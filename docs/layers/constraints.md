This page is normative and defines ADM semantics.

# Constraints

## Definition
Non-structural limitations (regulatory, organizational, technical).

## Allowed Content
- Regulatory limits
- Organizational limits
- Non-structural technical limits

## Forbidden Content
- Structural architecture definitions
- Layer substitution
- Deployment unit decomposition

## Common Violations
- Encoding structure as a constraint
- Redefining layer semantics through constraints
