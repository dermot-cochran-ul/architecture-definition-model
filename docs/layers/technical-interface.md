This page is normative and defines ADM semantics.

# Technical Interface

## Definition
Technology-specific bindings (protocols, databases, runtimes).

## Allowed Content
- Protocol bindings
- Database interface definitions
- Runtime-specific integration bindings

## Forbidden Content
- Business intent statements
- Layer redefinitions
- Non-technical capability descriptions

## Common Violations
- Replacing service contracts with technology internals
- Defining architecture structure in technology bindings
