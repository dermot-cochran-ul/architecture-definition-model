# Principles

## Single Source of Truth
The ADM defines architecture.

## Layer Discipline
Layers MUST be used exactly as defined.

## Semantic Integrity
Each ADM concept has exactly one meaning.

## Authority and Change Control
Architecture changes MUST be reflected in the ADM.
