# Tutorials

## Purpose
Tutorials explain how to read and apply existing ADM semantics without redefining them.

## Tutorial Set
1. Identify the correct ADM layer for an architectural statement.
2. Verify layer discipline for vertical and horizontal layers.
3. Validate that derived artifacts do not redefine ADM semantics.

## Non-Negotiable Rules for Tutorials
- Tutorials MUST reference existing ADM definitions.
- Tutorials MUST NOT introduce new ADM semantics.
- Tutorials MUST NOT collapse, merge, or skip layers.
